Приєднатись
Як це працює

1. Створіть сесію → покажіть QR партнеру
2. Обидва обмінюються ключами автоматично
3. Скануйте QR один одного (захист від MITM)
4. Оберіть файл → зашифровано → передано
5. Сервер бачить лише зашифровані дані
